# homebrew-tap

Homebrew tap for [CosmicMunkey](https://github.com/CosmicMunkey) apps.

## Install GRPC Nimbus

```bash
brew install --cask CosmicMunkey/tap/grpc-nimbus
```

Or tap first then install:

```bash
brew tap CosmicMunkey/tap
brew install --cask grpc-nimbus
```

## Apps

| Cask | Description |
|------|-------------|
| `grpc-nimbus` | Cross-platform desktop gRPC client with protoset support |
